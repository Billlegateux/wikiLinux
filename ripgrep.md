# Utilisation de ripgrep

Lien : https://github.com/BurntSushi/ripgrep



## Installation

```shell
sudo apt update
sudo apt install ripgrep
```



## Utilisation

```shell
# Pour rechercher un texte dans un fichier
rg mot fichier.txt

# Pour rechercher un texte dans une arborescence (en r√©cursif)
```
