# Naviguer dans l'arboresence

## Pour lister les dossiers/fichiers
```bash
ls -lha
exa --tree --long
```

## Pour naviguer
```bash
cd <nom du dossier>
# Pour remonter
cd ..
# Pour aller dans le dossier par défaut
cd
# Pour aller dans le dossier précédent
cd -
```
## Pour savoir le dossier où l'on se trouve
```bash
pwd
```
